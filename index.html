Akari Game

In this project, I created a version of Akari using JavaFX. The game included the ability to transition between levels, reset the puzzle, and a win screen. I used the Model/View/Controller archetype to create this program.

Image Rotation

This is a sample of my current coursework in Scientific Computing. In this Python-powered project, I import an image and create a large border surrounding it. I then utilize a matrix multiplication algorithm I derived for the project to rotate the image by a user-given amount. Feel free to check out my written report in more detail here.

Machine Code

As my concentration is in Systems, I have developed experience in lower-level programming languages such as C and machine code for Simple As Possible (SAP) and MIPS systems. This program, although simple in a high-level language, utilizes assembly code to print out all possible binary integers in a given length between 0 and 16. While programming in assembly is not extremely transferable to modern software development, my coursework in assembly code has given me a strong background and understanding of how computers receive, store, and manipulate information.

Gacha Game

In this JavaFX project, I worked alongside a group to create a game with a level and character select, automatic battling system, and gacha rolling system with bad luck protection. I was responsible for a large part of the character design and all of the logic surrounding the auto-battling. This project gave me extremely valueable experience in writing code in a collaborative setting. I learned the importance of specific documentation, code reviews, and design meetings as I experienced them firsthand while the game was in development.
